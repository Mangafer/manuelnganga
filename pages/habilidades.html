Linguagens

Linguagem de programação de propósito geral e de nível médio, conhecida por sua eficiência e amplamente utilizada em sistemas operacionais e aplicativos de sistemas.

Linguagem de programação orientada a objetos desenvolvida pela Microsoft como parte da plataforma .NET, usada para desenvolvimento de aplicativos Windows, jogos com Unity e muito mais.

Linguagem de estilo usada para estilizar páginas HTML e XML, define a apresentação visual dos elementos na página da web.

Linguagem de marcação usada para criar e estruturar páginas da web, define a estrutura e o conteúdo da página.

Linguagem de programação de alto nível e de propósito geral usada para criar páginas da web dinâmicas e interativas, executada no lado do cliente.

Sistema de gerenciamento de banco de dados relacional de código aberto, usado para armazenar e gerenciar dados em aplicativos da web.

Linguagem de script de código aberto usada principalmente para desenvolvimento web do lado do servidor, integrada com HTML para criar aplicativos da web dinâmicos.

Linguagem de consulta usada para interagir com bancos de dados relacionais, permite operações como inserção, atualização, exclusão e consulta de dados.
